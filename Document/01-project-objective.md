
# Project Objective

## Business Scenario

Latency IT Solutions is expanding its cloud operations and requires a secure identity and access management framework that supports multiple engineering teams while maintaining centralized administration and security.

As the Cloud Engineer responsible for the AWS environment, the objective of this project was to design and implement a scalable identity management solution using AWS Organizations, AWS IAM, and AWS IAM Identity Center.

The implementation focuses on creating a secure multi-account structure, defining user roles through permission sets, organizing users into groups, and applying the principle of least privilege to control access across the organization.

This project demonstrates how enterprise organizations can manage cloud identities consistently while reducing administrative overhead and improving security governance.
